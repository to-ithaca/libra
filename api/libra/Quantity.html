Quantity 

case class Quantity[A, I <: Singleton with Int, D <: Dimensions[_, _]](value: A) extends Product with Serializable

Represents a quantity in standard index form

A

the Numeric type of the quantity e.g. Int, Float, Double

I

the standard index form exponent

D

the dimensions

value

the standard form coefficient

Example: 

  1. scala> import spire.implicits._
scala> import libra._, libra.si._

scala> Quantity[Double, 3, Dimensions[Length :: HNil, HNil]](5.5) // represents 5.5 x 10^3 m
scala> res0: Quantity[Double, 3, Dimensions[Length :: HNil, HNil]] = Quantity(5.5)
