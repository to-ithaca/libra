Dimensional analysis for Scala

View on GitHub

Libra is a dimensional analysis library based on shapeless, spire and singleton-ops. It contains out of the box support for SI units for all numeric types.

To Use

Add this to your build.sbt:

resolvers += Resolver.bintrayRepo("to-ithaca", "maven")

libraryDependencies += "com.ithaca" %% "libra" % "0.1.0"

TLDR

Example usage:

scala> import spire.implicits._
import spire.implicits._

scala> import libra._, libra.si._
import libra._
import libra.si._

scala> (3.m + 2.m).show
res0: String = 5 * 10^0 m^1 [L^1]

scala> (3.m * 2.m).show
res1: String = 6 * 10^0 m^2 [L^2]

scala> (1.0.km + 2.0.m + 3.0.mm).show
res2: String = 1.002003 * 10^3 m^1 [L^1]

scala> (3.0.s / 3.0.ms).show
res3: String = 1.0 * 10^3  []

scala> //this should fail
     | 3.m + 2.kg
<console>:23: error: These quantities can't be added!  Most likely they have different dimensions.  If not, make sure that there's an implicit AdditiveSemigroup in scope.
Left: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]]
Right: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Mass :: shapeless.HNil,shapeless.HNil]]
       3.m + 2.kg
           ^

Why?

When we deal with numeric quantities, we often resort to Int, Double or Float types. These are incommunicative and error prone.

scala> val distance = 3.0 // 3 m
distance: Double = 3.0

scala> val time = 2.0 // 2 s
time: Double = 2.0

scala> val speed = distance + time // Oh no!
speed: Double = 5.0

There’s a mistake in our formula, but we won’t know without a decent set of tests.

Libra provides a Quantity which wraps base numeric types. It supports compile time dimensional analysis.

scala> import spire.implicits._
import spire.implicits._

scala> import libra._, libra.si._
import libra._
import libra.si._

scala> val distance = 3.0.m
distance: libra.QuantityOf[Double,Int(0),libra.si.package.Length] = Quantity(3.0)

scala> val time = 2.0.s
time: libra.QuantityOf[Double,Int(0),libra.si.package.Time] = Quantity(2.0)

scala> val speed = distance + time
<console>:32: error: These quantities can't be added!  Most likely they have different dimensions.  If not, make sure that there's an implicit AdditiveSemigroup in scope.
Left: libra.Quantity[Double,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]]
Right: libra.Quantity[Double,Int(0),libra.Dimensions[libra.si.package.Time :: shapeless.HNil,shapeless.HNil]]
       val speed = distance + time
                            ^

scala> val speed = distance / time // Yay!
speed: libra.Quantity[Double,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,libra.si.package.Time :: shapeless.HNil]] = Quantity(1.5)

scala> speed.show
res6: String = 1.5 * 10^0 m^1  s^-1 [L^1  T^-1]

Operations

Addition

Here is how you add quantities:

scala> val q = 3.m + 2.m
q: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]] = Quantity(5)

scala> q.show
res7: String = 5 * 10^0 m^1 [L^1]

You can add quantities for any kind of number type, provided there’s an AdditiveSemigroup for them. Here we’re using spire’s rationals:

scala> val q = r"1/3".m + r"2/3".m
q: libra.Quantity[spire.math.Rational,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]] = Quantity(1)

scala> q.show
res8: String = 1 * 10^0 m^1 [L^1]

But you can only add quantities of the same dimension:

scala> val q = 3.m + 4.kg
<console>:30: error: These quantities can't be added!  Most likely they have different dimensions.  If not, make sure that there's an implicit AdditiveSemigroup in scope.
Left: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]]
Right: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Mass :: shapeless.HNil,shapeless.HNil]]
       val q = 3.m + 4.kg
                   ^

scala> q.show
res9: String = 1 * 10^0 m^1 [L^1]

Subtraction is just the same:

scala> val q = 3.m - 2.m
q: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,shapeless.HNil]] = Quantity(1)

scala> q.show
res10: String = 1 * 10^0 m^1 [L^1]

Multiplication

Here’s how you multiply quantities:

scala> val q = 3.m * 2.kg
q: libra.Quantity[Int,Int(0),libra.Dimensions[libra.si.package.Length :: libra.si.package.Mass :: shapeless.HNil,shapeless.HNil]] = Quantity(6)

scala> q.show
res11: String = 6 * 10^0 m^1 kg^1 [L^1 M^1]

You can also divide quantities:

scala> val q = 3.0.m / 2.0.kg
q: libra.Quantity[Double,Int(0),libra.Dimensions[libra.si.package.Length :: shapeless.HNil,libra.si.package.Mass :: shapeless.HNil]] = Quantity(1.5)

scala> q.show
res12: String = 1.5 * 10^0 m^1  kg^-1 [L^1  M^-1]

Power

Here’s how you raise quantities to a power:

scala> val q = 3.0.m^(3)
q: libra.Quantity[Double,Int(0),libra.Dimensions[libra.si.package.Length :: libra.si.package.Length :: libra.si.package.Length :: shapeless.HNil,shapeless.HNil]] = Quantity(27.0)

scala> q.show
res13: String = 27.0 * 10^0 m^3 [L^3]

Inverse

You can also invert quantities:

scala> val q = 3.0.m.invert
q: libra.Quantity[Double,Int(0),libra.Dimensions[shapeless.HNil,libra.si.package.Length :: shapeless.HNil]] = Quantity(0.3333333333333333)

scala> q.show
res14: String = 0.3333333333333333 * 10^0 m^-1 [L^-1]

Extracting the base value

Here’s how you extract the value as a numeric in base units:

scala> 3.km.baseValue
res15: Int = 3000

Rolling your own

Let’s say we want to code up a game of Catan. The base SI units don’t cut the mustard, so let’s make our own. Our base dimensions are going to be Stone, Wood, Wheat, Sheep and Time.

scala> import libra._, libra.ops.base._
import libra._
import libra.ops.base._

scala> object catan {
     |   type Stone
     |   type Wood
     |   type Wheat
     |   type Sheep
     |   type Time
     |   
     |   //we ought to provide shows
     |   implicit def stoneShowDimension: ShowDimension[Stone] = ShowDimension[Stone]("S")
     |   implicit def woodShowDimension: ShowDimension[Wood] = ShowDimension[Wood]("W")
     |   implicit def wheatShowDimension: ShowDimension[Wheat] = ShowDimension[Wheat]("F")
     |   implicit def sheepShowDimension: ShowDimension[Sheep] = ShowDimension[Sheep]("B")
     |   implicit def timeShowDimension: ShowDimension[Time] = ShowDimension[Time]("T")
     |   
     |   implicit def stoneShowUnit: ShowUnit[Stone] = ShowUnit[Stone]("Rocks")
     |   implicit def woodShowUnit: ShowUnit[Wood] = ShowUnit[Wood]("Logs")
     |   implicit def wheatShowUnit: ShowUnit[Wheat] = ShowUnit[Wheat]("Bushel")
     |   implicit def sheepShowUnit: ShowUnit[Sheep] = ShowUnit[Sheep]("Sheep")
     |   implicit def timeShowUnit: ShowUnit[Time] = ShowUnit[Time]("Turn")
     |   
     |   
     |   //implicit conversions would be useful too
     |   implicit final class CatanOps[A](val a: A) extends AnyVal {
     |      def wood: OneOf[A, Wood] = Quantity(a)
     |      def stone: OneOf[A, Stone] = Quantity(a)
     |      def sheep: OneOf[A, Sheep] = Quantity(a)
     |      def turns: OneOf[A, Time] = Quantity(a)
     |   }
     | }
defined object catan

scala> import spire.implicits._
import spire.implicits._

scala> import catan._
import catan._

scala> val tradingRate = 4.0.wood / 1.0.sheep
tradingRate: libra.Quantity[Double,Int(0),libra.Dimensions[catan.Wood :: shapeless.HNil,catan.Sheep :: shapeless.HNil]] = Quantity(4.0)

scala> val sheeple = 8.0.wood / tradingRate
sheeple: libra.Quantity[Double,Int(0),libra.Dimensions[catan.Sheep :: shapeless.HNil,shapeless.HNil]] = Quantity(2.0)

scala> sheeple.show
res16: String = 2.0 * 10^0 Sheep^1 [B^1]
